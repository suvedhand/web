This website uses the following platforms and frameworks:

  1. Hugo - open source static site generator
  2. Hugo Tranquilpeak Theme - a custom template to personolize the website
  3. R Studio Desktop - an integrated development Environment (IDE) for R
  4. R Blogdown - open source R package to generate static websites based on R and Hugo
  5. R Markdown to write, analyze and share posts
  6. Github for hosting the website code and version control
  7. Netlify to deploy using continuous integration with Github
